# GeoHub Open

Open Geoscience code hub from GeoVis System, Jeonbuk National University

## Overview

GeoHub Open is the public-facing repository for verified, publication-ready geoscience codes from the GeoHub ecosystem. This repository contains carefully selected tools and programs that have been developed, tested, and documented for community use.

## Purpose

- **Open Science**: Share verified geoscience tools with the research community
- **Publication Support**: Accompany scientific publications with reproducible code
- **Community Contribution**: Provide useful tools for geoscience researchers worldwide

## What's Included

This repository contains select programs from:
- **GeoStream**: Refactored and validated research codes
- **GeoSpring**: Production-ready custom tools
- **GeoWave**: Framework components suitable for public use
- **GeoTool**: Useful utilities for the geoscience community

## Quality Assurance

All code in GeoHub Open has:
- ✅ Complete testing and validation
- ✅ Comprehensive documentation (English)
- ✅ Reproducible examples
- ✅ Performance benchmarks
- ✅ Publication-quality readiness

## Using This Repository

### Installation
```bash
git clone https://github.com/Geoverse-lab/geohub_open.git
cd geohub_open
pip install -r requirements.txt
```

### Documentation
Each program includes:
- README with overview and usage
- Parameter specifications
- Theoretical background
- Examples and tutorials
